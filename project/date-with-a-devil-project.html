Date With A Devil (Game Jam)

Game made for Ludum Dare 43

Date with a Devil is a game jam game made for the Ludum Dare 42 game jam in 48 hours. It was made in Unity and released as a web game on itch.io, you can play it online here. It is a dialogue driven game show where you play as a contestant trying to win the affections of the demon your cult worships, with the ultimate prize being chosen as the sacrifice to the demon. This odd premise was devised to fit with the theme of the game jam which was “Sacrifices must be made”, we used this theme as the name of our in-game game show. It is a multiplayer game and can be played with 1-4 players, any non-human slot is controlled by a computer AI player.

My primary responsibilities included:

  • System to read in dialogue scripts and display them to the screen with the necessary effects such as slow text reveal and speech bubble carot location
  • System to run the core game show loop of the dialogue, including showing questions to the player during trivia and processing the performance of each player during these segments
  • Setting up multiplayer capabilities
  • AI behaviors for game show questions being asked
